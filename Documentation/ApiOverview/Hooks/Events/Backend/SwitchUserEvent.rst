===============
SwitchUserEvent
===============

This event is dispatched when a "SU" (switch user) action has been triggered.

API
---


 - :Method:
         getSessionId()
   :Description:
         Returns the session ID of the current session.
   :ReturnType:
         string


 - :Method:
         getTargetUser()
   :Description:
         Returns the target user record.
   :ReturnType:
         array


 - :Method:
         getCurrentUser()
   :Description:
         Returns the current user record.
   :ReturnType:
         array
