=======================
AfterFolderRenamedEvent
=======================

This event is fired after a folder was renamed.
*Examples*: Add custom processing of folders or adjust permissions.

API
---


 - :Method:
         getFolder()
   :Description:
         Returns the folder after renaming.
   :ReturnType:
         \TYPO3\CMS\Core\Resource\Folder
