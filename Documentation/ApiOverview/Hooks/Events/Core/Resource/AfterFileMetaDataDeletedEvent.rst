=============================
AfterFileMetaDataDeletedEvent
=============================

This event is fired once all metadata of a file was removed, in order to manage custom metadata that was
added previously.

API
---


 - :Method:
         getFileUid()
   :Description:
         Get the file uid of the current file.
   :ReturnType:
         int
