.. _ModifyUrlForCanonicalTagEvent:


=============================
ModifyUrlForCanonicalTagEvent
=============================

Event to alter (or remove) an URL for the href="" attribute of a canonical URL.

API
---


 - :Method:
         getUrl()
   :Description:
         Returns the currently set canonical URL.
   :ReturnType:
         string


 - :Method:
         setUrl(string $url)
   :Description:
         Set (overwrite) the canonical URL.
   :ReturnType:
         void
